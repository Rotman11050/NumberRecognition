# Handwritten Digit Recognizer
A Python application that uses a Convolutional Neural Network (CNN) to recognize hand-drawn digits using the MNIST dataset.

## Purpose of the project
Learning TensorFlow and getting better with other features

## Features
- Train a CNN model using TensorFlow
- Predict handwritten digits
- Image preprocessing with OpenCV
- Confidence score for predictions
- MNIST dataset integration
- GUI with Tkinter(Soon)

## Technologies Used
- Python
- TensorFlow
- OpenCV
- NumPy
- ImageIO
- Tkinter(Will be included)

## Installation

```bash
pip install tensorflow opencv-python numpy imageio
```

## Workflow
1. Load MNIST dataset
2. Normalize image values
3. Train CNN model
4. Preprocess input image
5. Predict digit and confidence
